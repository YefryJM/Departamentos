¿Qué es un Protocolos HTTP?

es el protocolo de transmisión de información de la World Wide Web, es decir, el código que se establece para que el computador solicitante y el que contiene la información solicitada puedan “hablar” un mismo idioma a la hora de transmitir información por la red

¿Qué es un Query String?

Query string o, en español, cadena de consulta es un término informático que se utiliza para hacer referencia a una interacción con una base de datos. Es la parte de una URL que contiene los datos que deben pasar a aplicaciones web como los programas CGI.

¿Qué es un Verbos HTTP?

Los verbos Http involucrados en un sistema REST son GET, POST, PUT, PATCH y DELETE. GET es el que usamos para consultar un recurso. Una de las principales características de una petición GET es que no debe causar efectos secundarios en el servidor, no deben producir nuevos registros, ni modificar los ya existentes.

¿Qué es un DNS?

El DNS, o sistema de nombres de dominio, traduce los nombres de dominios aptos para lectura humana por ejemplo, www.amazon.com a direcciones IP aptas para lectura por parte de máquinas

¿Qué es un CDN?

es una red de servidores que distribuye contenido de un servidor de "origen" a todo el mundo, almacenando ese contenido en caché en un lugar más cercano al de cada usuario final que accede a Internet con un dispositivo habilitado para la Web.
